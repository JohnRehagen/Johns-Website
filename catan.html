Settlers of Catan is one of my favorite board games. I have great memories of playing it with my family around our kitchen table and with friends. I built this calculator as a thought experiment - using my VBA skills to apply expected value and game theory into an interactive playgrond to understand the game better.

The object of the game is to gather more resources than your opponents and use the resources to grow your settlement faster than them in order to win.

The user inputs the tile resource and number for each tile before pressing the "run calculation" button, as shown in the gif below.

The VBA code then calucates the expected value of each node, where players can place pieces, and breaks them down by resource. It also populates the graphical representation of the board, helping the user to easily relate the calculation to the real world.

The far right table allows the player to compare the strength of each node, including the strength of each adjacent resource. You'll also notice that each node is populated with a color that represents it's strength, helping correlate node name to actual location on the board.

The middle-bottom table allows the player to see the relative strength of each resource according to the specific board setup. As the setup is random each game and there are a finite number of resources, often games will have different balances of resources in play. This allows the player to quickly know what will be prevalent and scarce in the game; helping them to make decisions accordingly.

Although it is just a fun application, this calculator is meant to demonstrate how powerful excel VBA is, and my competency with it. I haven't mentioned this yet, but I only spent about a day producing this project.
